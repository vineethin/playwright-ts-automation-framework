# Playwright TypeScript Automation Framework (Starter)

A production-style Playwright + TypeScript test automation framework with:
- Page Object Model (POM)
- Fixtures for dependency injection (page objects per test)
- Multi-browser execution (chromium/firefox/webkit)
- Tag-based runs (smoke/regression)
- HTML reporting + test artifacts (screenshots/videos/traces)
- Optional auth session reuse via storageState (supports non-SSO and SSO patterns)

## Prerequisites
- Node.js LTS
- VS Code (recommended)

## Quick Start
```bash
npm install
npx playwright install
